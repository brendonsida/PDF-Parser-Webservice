##Unit Test File Structure:

```
├── test-suite/
|   |
│   ├── tests
|   |   |
│   │   ├── <test-num>
│   │   │   ├── inputs
│   │   │   |   ├── coordinates
│   │   │   |   ├── outputtype
│   │   │   |   ├── testfile
│   │   │   ├── outputs
│   │   │   |   ├── out.correct.csv
```

**For Example:**

```
├── test-suite/
|   |
│   ├── tests
|   |   |
│   │   ├── 1
│   │   │   ├── inputs
│   │   │   |   ├── coordinates
│   │   │   |   ├── outputtype
│   │   │   |   ├── testfile
│   │   │   ├── outputs
│   │   │   |   ├── out.correct.csv
│   │   ├── 2
...
```
