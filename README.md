## Building and Running:

**For Clean Build:**

    cd _maven/webserver
    mvn clean install
    
**To Start WebServices:**

    cd _maven/webserver 
    ./run_server.sh
    
The webservice runs on localhost:8080.

**To Run Test Suite:**

    cd test-suite
    ./test_runner

**To Clean Compiled Files and Keep Tabula Compiled Files**

    make clean

**To Clean Entire Project:**

    make cleaninstall
