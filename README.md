Team Front-End (Alex, Ben, Brendon, Tim)

## Building and Running:

**For Clean Build:**

    cd _maven/webserver
    mvn clean install

**To Start WebServices:**

    cd _maven/webserver 
    java -jar ./target/team.frontend.app-jar-with-dependencies.jar

The webservice runs on localhost:8080.

**To Clean:**

    mvn clean
